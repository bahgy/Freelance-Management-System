
Freelance Management System 🚀

A modern Freelance Project Management System built using ASP.NET Core Web API, designed to help freelancers and small teams organize projects, tasks, clients, notifications, and AI-powered insights.

The system also supports future integration with:

🤖 Telegram Bot
⚡ n8n Automation
🧠 AI Insights (Gemini / OpenAI)
📊 Dashboard Analytics
📌 Project Goal

This platform helps freelancers and teams:

Manage projects professionally
Track tasks and deadlines
Organize clients
Receive smart AI insights
Automate workflows using n8n
Scale later into a full SaaS platform
